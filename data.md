# link ou instruções para baixar o dataset do Kaggle

### O dataset utilizado é público e está disponível no Kaggle:
🔗 https://www.kaggle.com/datasets/snap/amazon-fine-food-reviews

Ele contém avaliações de produtos alimentícios com:

- Texto completo da avaliação
- Nota (Score) de 1 a 5
- Informações do usuário e do produto

Para o desenvolvimento inicial, foi utilizada uma amostra de 10.000 registros.
